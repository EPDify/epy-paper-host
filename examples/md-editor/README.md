# EPY Markdown Editor

This is a clean, easy-to-use Single Page Application (SPA) designed for writing, viewing, and editing Markdown (`.md`) files. It functions perfectly both as a standalone local web application and as an integrated editor for remote files.

## Key Features

- **Live Preview Environment:** Watch your formatting instantly come to life as you type. You can switch between "Source Mode" (just the code), "Preview Mode" (just the document), or "Split Mode" (both side-by-side).
- **Drag-and-Drop Friendly:** Quickly upload and edit any Markdown file from your computer by simply dragging and dropping it into the browser window.
- **Local Download:** When you are done drafting or editing your document, you can instantly download it back to your local computer with a single click.
- **Remote Editing & Saving:** If provided a file path in the web address, the editor will load the file directly from the host device. You can then make changes and use the "Submit" button to save the file right back onto the device's storage.
- **Markdown Cheat Sheet:** Forgotten how to create a list or format a link? Use the built-in, quick-reference guide (accessible via the info button) to jog your memory without leaving the editor.

## How It Works 
- **Standalone Mode:** Simply open the editor. Drag in a file or click "Start Blank". You can write and download your finished product.
- **File Manager Mode:** If the Markdown file is opened directly from the File Manager, the editor will pull the document out of the File Manager. Clicking "Submit" pushes any modifications back to the File Manager.
