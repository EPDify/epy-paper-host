# EPY PDF Editor

This robust Single Page Application (SPA) allows you to edit, manage, and annotate PDF files directly in your web browser. All processing is done locally on your device, ensuring maximum privacy and speed without the need to send your private documents to external cloud servers.

## Key Features

- **Document Annotation:** Draw free-hand, add custom text fields, highlight important sections, or draw rectangle shapes directly over your PDF pages.
- **Merge & Combine:** Have multiple PDFs? Just add them all into the editor, and you can merge them into one seamless document. 
- **Page Management:** Add blank pages into your document as needed, right from the sidebar. You can also view thumbnails of all pages in your document for easy navigation.
- **Secure Redaction:** Selectively hide and permanently scrub sensitive information from your documents using the Redact tool. Once redactions are applied and exported, the restricted content cannot be recovered by viewers.
- **Full Control:** Navigate documents comfortably using the Hand tool, select and delete specific annotations, zoom in/out for detailed work, and instantly Undo or Redo recent changes.
- **Light & Dark Themes:** Work comfortably in any environment using the built-in Light and Dark Mode toggle.

## How it Integrates

- **Drag & Drop Workflow:** You can simply open the web-app and drag a PDF onto the interface to get started immediately.
- **Download or Save:** Once you've completed your edits, use the **Download** button to instantly snap off a copy of the edited PDF to your local computer. 
- **File Manager Mode:** If the PDF file is opened directly from the File Manager, the editor will pull the document out of the File Manager. Clicking "Upload" pushes any modifications back to the File Manager.
